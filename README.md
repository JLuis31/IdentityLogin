
# IdentityLogin

Login application using Identity and JWT


## Run Locally

Clone the project

```bash
  git clone https://github.com/JLuis31/IdentityLogin.git
```

Go to the project directory

```bash
  cd IdentityLogin
```



## Tech Stack


**Server:** C#, .NET

## Features

- Api Post
- Api Get
- Api Delete
- Usage of JWT and RefreshTokens with Identity
